# CryptoPortfolio V0 (MVP)

Minimal Viable Product for the CryptoPortfolio application.
Allows adding and listing assets with local persistence.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
# Add an asset
python -m crypto_portfolio.main add BTC 0.5 25000

# List assets
python -m crypto_portfolio.main list
```
